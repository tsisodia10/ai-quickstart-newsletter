# Red Hat AI Quickstarts Newsletter

A visually stunning, editorial-style single-page newsletter website built for **Red Hat's AI Quickstarts** initiative, showcased at **Red Hat Summit 2026**. Designed with modern tech-magazine typography, bold Red Hat brand aesthetics, and a clean light theme.

---

## 🌟 Overview

The **Red Hat AI Quickstarts Dispatch** highlights 17 production-grade, open-source AI use cases built for fast deployment on **Red Hat OpenShift AI**. From anti-money laundering workflows to IT automation and prompt security, these quickstarts enable enterprise teams to move from zero to production-ready AI in hours, not months.

### Key Sections Included:

1. **Header Dispatch Bar**: Issue details, timestamp, and location tag.
2. **Hero Section**: Bold typographic headline, Red Hat red diagonal stripe, and call to action.
3. **From the Team (Editor's Note)**: Editorial welcome message from the AI Quickstarts Core Team.
4. **News Ticker Strip**: Continuous marquee banner highlighting major Summit 2026 announcements.
5. **Mission Statement & Stat Cards**: Key metrics covering Quickstarts, Industries, and Platform.
6. **Summit 2026 Spotlight**: Dramatic feature card spotlighting the **Lemonade Stand Assistant + Guardrails** live demo.
7. **Complete 17 Quickstarts Grid**: Sleek card layout highlighting persona-targeted hooks (*"Who it's for" & "Why it matters"*), industry pills, estimated deployment times, and direct links to official documentation.
8. **As Seen at Red Hat Summit 2026**: High-impact pull-quote section detailing OpenShift AI, vLLM inference, and Model Context Protocol (MCP).
9. **Get Started & Call to Action**: 3-step deployment workflow with direct links to documentation and GitHub repositories.
10. **Footer**: Official branding, quick links, and Red Hat Summit 2026 event details.

## 🎨 Brand & Design System

- **Color Palette**:
  - **Red Hat Red**: `#EE0000` (Hero accents, buttons, key borders)
  - **Deep Black**: `#0D0D0D` (Headings, high-contrast dark sections)
  - **Warm Off-White**: `#F5F3EE` (Section backgrounds, callouts)
  - **Crisp White**: `#FFFFFF` (Main background & card canvases)
  - **Industry Badges**: Custom pastel pills for Banking (Gold), Healthcare (Green), Retail (Blue), and Enterprise (Red).
- **Typography**:
  - **Bebas Neue**: Display headlines & high-impact titles
  - **Playfair Display**: Editorial quotes & section subtitles
  - **IBM Plex Mono**: Technical callouts, tags, & code blocks
  - **Inter**: Crisp, readable body typography
- **Animations**: Scroll-triggered reveal effects using `IntersectionObserver` and smooth CSS marquee keyframes.

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
